# Book Club

#### App to store and manage profiles for members of Book Club!

#### By **Maria Del Castillo**

## Description

On this app, you can see profiles for all the members of book club, including books they've suggested to the group, when they've hosted in the past, and even any dietary restrictions they may have so you can prepare snacks wisely.


## Setup/Installation Requirements
1. Clone this repository.
2. In the terminal, navigate into the repository and run the following commands:
  $ npm install
  $ bower install
3. Create file 'api-keys.ts' in the app folder.
4. Create a firebase account at https://firebase.google.com/ . On their site choose 'Add Firebase to your web app'.
5. Place the code snippet they provide in the 'api-keys.ts' file you created.
6. Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.
